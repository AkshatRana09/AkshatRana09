- Automated **SMS alerts via Twilio API** within 3 seconds of detection
- Live web dashboard with **real-time visibility scores & camera feed**

`Python` `OpenCV` `Flask/FastAPI` `Twilio API` `PostgreSQL` `HTML/CSS/JS`
